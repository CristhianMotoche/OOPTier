# Introducción
Un lenguaje es un conjunto e oraciones válidas. ¿Qué hace que una oración sea válida?
La validez depende de dos cosas: sintaxis y semántica.
A language is a set of valid sentences. What makes a sentence valid? You can break
validity down into two things: syntax and semantics.

# Sintaxis
Hace referencia a la estructura gramática.

# Semantica
Se refiere al significado de los símbolos del vocabulario organizados dentro de
esa estructura.

# Diferencias:
- La sintaxis sigue una estructura definida, la semántica no.
- Una sentencia puede tener el mismo significado (semántica), pero no necesariamente
la misma estructura (sintaxis).
- Si la sintaxis es errónea, el programa no se ejecutará. Una sintaxis correcta
no implica que un programa sea semánticamente correcto.
